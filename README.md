# Student Registration Status Checker

 Purpose
This project implements a registration validation tool across four major programming languages. It checks if a student's unit load exceeds the standard limit of 7 units.

 How to Run
 Python
Run: `python python/main.py`

 Java
Compile: `javac java/StatusChecker.java`  
Run: `java -cp java StatusChecker`

 JavaScript (Node.js)
Run: `node javascript/index.js`

 C
Compile: `gcc c/main.c -o checker`  
Run: `./checker`

What this demonstrates
This project highlights the differences in syntax, memory management, and execution models (Compiled vs. Interpreted) between C, Java, Python, and JavaScript.# Student-Registration-Status-Checker
